Navigation in HTML

Navigation in HTML is essential for creating user-friendly websites. The <nav> tag is specifically designed to define a set of navigation links. This tag helps in organizing and presenting links for user navigation, enhancing accessibility by aiding screen readers in identifying primary navigation areas.

Basic Usage of <nav> Tag 
                <!DOCTYPE html>
                <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>HTML nav Tag</title>
                        </head>
                        <body>
                            <h2>HTML nav Tag</h2>
                        <nav>
                            <a href="#">Home</a> 
                            <a href="#">Interview</a>
                            <a href="#">Languages</a> 
                            <a href="#">Data Structure</a> 
                            <a href="#">Algorithm</a>
                        </nav>
                    </body>
                </html>
