Table Tag

Basic Table Formate

Syntax

        <table>
            <tr>
                <th>Name</th>
                <th>Age</th>
            </tr>
            <tr>
                <td>Alice</td>
                <td>24</td>
            </tr>
        </table>

Output

Name Age
Akshay 22

Important Tags

Syntax

        <table border="1" style="text-align: center;">
            <tr>
                <th>Tag</th>
                <th>Description</th>
            </tr>
            <tr>
                <td><table></td>
                <td>Defines a table</td>
            </tr>
            <tr>
                <td><tr></td>
                <td>Table row</td>
            </tr>
            <tr>
                <td><th></td>
                <td>Table header (bold, centered)</td>
            </tr>
            <tr>
                <td><td></td>
                <td>Table data cell</td>
            </tr>
    </table>

Output

Tag Description
<table> Defines a table
<tr> Table row
<th> Table header (bold, centered)
<td> Table data cell

Table Example with <caption>

1] Table Example without <caption>

Without <caption> : The table has No Title; only headers and data are shown.

NAME AGE CLASS RESULT
Yash 22 12th PASS
Kedar 22 10th PASS
Rohit 22 12th FAIL
King 20 12th FAIL

2] Table Example with <caption>

With <caption> : The table has a Descriptive Title above (or below) the table, improving accessibility and clarity.
