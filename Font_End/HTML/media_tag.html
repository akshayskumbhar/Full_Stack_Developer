iFrame tag

What is an iFrame?
An iFrame (inline frame) is an HTML element that allows you to embed another HTML document within the current HTML document. This is done using the <iframe> tag. iFrames are commonly used to embed content such as videos, advertisements, maps, or other interactive elements from external sources into a web page.
How to Use iFrame
To use an iFrame, you need to include the <iframe> tag in your HTML document and specify the source (src) attribute, which is the URL of the content you want to embed. Here is a basic example of an iFrame embedding a YouTube video:

Syntax :-

<iframe src="../HTML/Media_Tag_files/images.html" width="90%" height="400px"></iframe>
