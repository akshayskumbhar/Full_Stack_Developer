    üìòDEFINATION :
            An Ordered List is used to display a sequence of items in a numbered format, 
            where the order matters (e.g., steps in a process, ranking, etc.).

        üî∏ Use the <ol> tag to define an ordered list.
        üî∏ Use <li> (list item) inside <ol> to define individual list elements.

        <li>Item1</li>
        <li>Item2</li>
