ORDERED LIST

    CONCEPT : 
            Ordered lists are used to Display Items in Ordered Way(Numbered Sequence).
            <ol> : Ordered_List
            <li> : List_Items
    
    SYNTAX :
            <ol>
                <li>Item1</li>
                <li>Item2</li>
            </ol>
    
    EXAMPLE : 
            <ol>
                <li>RAM</li>
                <li>SHYAM</li>
                <li>GHANSHYAM</li>
                <li>BABURAO</li>
                <li>GANAPAT</li>
                <li>DAGADU</li>
            </ol>

    OUTPUT :
  1. PAPITA
  2. MAYA
  3. ARUNDHATI
  4. GANAPA
  5. BUNTY
  6. RAMESH
