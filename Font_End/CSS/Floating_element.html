2] Z-index

        ‚úÖ Concept 
        
            ‚óè	z-index controls the stacking order of positioned elements (which element appears on top).

            ‚óè	Only works when the element has a position other than static (i.e., relative, absolute, fixed, sticky).

            ‚óè	Higher z-index = element appears in front.

            ‚óè	Default stacking order is determined by the DOM order, unless overridden using z-index.


üß© Syntax
        .box1 {
            position: absolute;
            top: 40px;
            left: 40px;
            width: 150px;
            height: 150px;
            background: rgba(255, 0, 0, 0.6);
            z-index: 1; /* lower ‚Üí behind */
            }

    .box2 {
            position: absolute;
            top: 80px;
            left: 80px;
            width: 150px;
            height: 150px;
            background: rgba(0, 0, 255, 0.6);
            z-index: 5; /* higher ‚Üí in front */
            }
