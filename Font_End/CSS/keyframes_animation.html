@Keyframes Property

Defination: CSS Animations allow elements to animate using keyframes, which specify multiple
            states over a duration. Animations can loop, reverse, pause, or run automatically

                
Syntax: 

                

    @keyframes slideUp {
                        0% { 
                            transform: translateY(100px); 
                            opacity: 0; 
                            }
                        100% { 
                            transform: translateY(0); 
                            opacity: 1; 
                            }
                        }

Output:

animation: fadeInUp 1s ease-out forwards;

Difference Between Ease-in Vs Ease-out

Ease-in Ease-out Ease-in-out
animation: Easei 1s ease-in forwards;
animation: Easeo 1s ease-out forwards;
animation: Easeo 1s ease-in-out forwards;

Difference Between Forword Vs Backword

Ease-in Forword Ease-in Backword Ease-out Forword Ease-out Backword
animation: easein 1s ease-in forwards;
animation: easeou 1s ease-out backword;
animation: easein 1s ease-in forwards;
animation: easeou 1s ease-out backword;
Ease-in-out for forwards and Backword
animation: easein 1s ease-in-out forwards;
animation: easein 1s ease-in-out backword;
