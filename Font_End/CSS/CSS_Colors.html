CSS Colors

  1. CSS supports a wide range of colors that can be used to style web pages.
  2. These colors can be Specified using color names, hexadecimal values, or RGB values.
  3. Following are the Different ways to specify colors in CSS.

2] By : HEX Values

  1. Hexadecimal triplet with 3 digits (#rgb).
  2. Each digit represents Red, Green, Blue (0-F in hex).
  3. Browser expands each digit to double (e.g. #f00 → #ff0000).
  4. Example :
      3 DIGIT HEX VALUE
    • #000 : #rgb : r=0 , g=0 , b=0
    • #fff : #rgb : r=f , g=f , b=f
    • #f00 : #rgb : red=f , g=0 , b=0
    • #0f0 : #rgb : r=0 , green=f , b=0
    • #00f : #rgb : r=0 , g=0 , blue=f

      • 6 DIGIT HEX VALUE
    • #000000 : #rrggbb : red=00 , green=00 , blue=00
    • #ff0000 : #rrggbb : red=ff , green=00 , blue=00
    • #00FF00 : #rrggbb : red=00 , green=FF , blue=00
    • #0000FF : #rrggbb : red=00 , green=00 , blue=FF

3] By : RGB Values

  1. Specifies color by mixing Red, Green, Blue values.
  2. Each channel carries: 0-255 integers Range.
  3. Syntax : property: rgb(R, G, B);
  4. Example : color: rgb(255,0,0);
    • rgb(0,0,0) : red=0 , green=0 , blue=0
    • rgb(255,255,255) : red=255 , green=255 , blue=255
    • rgb(255,0,0) : red=255 , green=0 , blue=0
    • rgb(0,255,0) : red=0 , green=255 , blue=0
    • rgb(0,0,255) : red=0 , green=0 , blue=255

4] By : RGBA Values

  1. Extension of RGB with Alpha channel.
  2. Alpha controls opacity→ 1.0 = fully opaque(Original Color), 0.0 = fully transparent.
  3. Syntax → property: rgba(R, G, B, A);
  4. Example :
    • rgba(255,0,0,1.0) → red=255 , green=0 , blue=0 , alpha=1.0
    • rgba(255,255,255,0.0) → red=255 , green=0 , blue=0 , alpha=0.0

5] By : HSL Values

  1. Uses Hue, Saturation, Lightness for intuitive color control.
  2. Hue (Value in degree): Range (0° - 360°) → 0/360=Red, 120=Green, 240=Blue
  3. Saturation (Value in %): Range (0% - 100%) → 100% (full color)
  4. Lightness (Value in %): Range (0% - 100%) → 50% (normal)
  5. Lightness : 0% (black) → 50% (normal) → 100% (white)
  6. Syntax : property: hsl(hue[0-360], saturation[0-100]%, lightness[0-100]%);
  7. Example :
    • hsl(360,100%,50%) : hue=360 deg° , saturation=100% , lightness=50%

6] By : HSLA Values

  1. Extension of HSL with Alpha channel.
  2. Alpha controls opacity: 1.0 = fully opaque(Original Color), 0.0 = fully transparent.
  3. Syntax : property: hsla(H, S, L, A);
  4. Example :
    • hsla(360,100%,50%,1.0) : hue=360 deg° , saturation=100% , lightness=50% , alpha=1.0
    • hsla(360,100%,50%,0.0) : hue=360 deg° , saturation=100% , lightness=50% , alpha=0.0
