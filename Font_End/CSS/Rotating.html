Rotating Animation

Syntax

                @keyframes rotate {
                    from {
                    transform: rotate(0deg);
                    }
                    to {
                    transform: rotate(360deg); /* full rotation */
                    }
                }
animation: rotate 3s linear infinite;

Rotating Circle Reverse Animation

Syntax

                @keyframes rotater {
                  from {
                    transform: rotate(360deg);
                  }
                  to {
                    transform: rotate(0deg); /* full reverse rotation */
                  }
                }
animation: rotate 3s linear infinite;
