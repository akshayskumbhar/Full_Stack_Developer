CSS Basics

Three Types Of CSS

1] Inline CSS : Used for Specific Element From the Web-Page.
Syntax: 
            index.html

                <div style="color: black; margin: 10;"></div>

2] Internal CSS : Used for Single Web-Page.
Syntax: 
            index.html

                <style> 
                    color: black; 
                    margin: 10;
                    padding: 10px;
                </style>

3] External CSS : Used for Multiple Web-Pages.
Syntax: 
                style.css

                .body{
                    background: linear-gradient(to left, red, blue)
                }
                div{
                    margin: 10px;
                    padding: 10px;
                    border: 5px solid black;
                }
