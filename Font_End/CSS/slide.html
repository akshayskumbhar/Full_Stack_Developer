Difference Between Left/Right Animation

Slide Left Slide Right Slide Both
animation: slideInLeft 1s ease-out infinite;
animation: slideInLeft 1s ease-out infinite;

Slide Left Animation

Syntax

                @keyframes slideInLeft {
                        0% {
                        transform: translateX(-100%); /* start outside left */
                        opacity: 0;
                        }
                        100% {
                        transform: translateX(0);
                        opacity: 1;
                        }
                    }
animation: slideInLeft 1s ease-out infinite;

Slide Right Animation

Syntax

                @keyframes slideInRight {
                        0% {
                        transform: translateX(0%); /* start inside left */
                        opacity: 1;
                        }
                        100% {
                        transform: translateX(100%);
                        opacity: 0;
                        }
                    }
animation: slideInLeft 1s ease-out infinite;

Slide Right And Left (Loop) Animation

Syntax

               @keyframes slideInLeftRight {
                        0% {
                        transform: translateX(0%); /* start outside left */
                        opacity: 0;
                        }
                        40% {
                        transform: translateX(50%); /* start inside center */
                        opacity: 1;
                        }
                        50% {
                        transform: translateX(50%); /* start inside center */
                        opacity: 1;
                        }
                        100% {
                        transform: translateX(100%); /* start exit right */
                        opacity: 0;
                        }
                    }
animation: slideInLeftRight 2s ease-out infinite;
