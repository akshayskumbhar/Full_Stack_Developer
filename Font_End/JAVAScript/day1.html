1. What is JavaScript?

Definition: JavaScript is a high-level, interpreted programming language used to make web pages interactive.

Use: Validation, animations, dynamic content, APIs.

2. Variables

Definition: Variables store data values.

Syntax:

3. Data Types

Definition: A function is a block of reusable code.
