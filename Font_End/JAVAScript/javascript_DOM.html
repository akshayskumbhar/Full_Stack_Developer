Javascript DOM Content Manipulation

1] innerHTML

Definition: innerHTML is used to get or set the HTML Content inside an element including tags

        Syntax: 
                element.innerHTML //get
                element.innerHTML = value //get
HelloWorld!

2] innerText

Definition: innerText returns only the visible text inside an element.

        Syntax: 
                element.innerText //get
                element.innerText = value //get
HelloWorld Myself Akshay Kumbhar!

3] textContent

Definition: terxtContent returns all text, including hidden elements.

        Syntax: 
                element.textContent //get
                element.textContent = value //get
Hello World!

4] outerHTML

Definition: outerHTML gets or replaces the entire element including itself.

        Syntax: 
                element.outHTML //get
                element.outerHTML = value //get
Original Element Box

5] outerText

Definition: outerText replaces the entire element with plain text.

        Syntax: 
                element.outerText = value //get
Hello Element Box

6] value Property

Definition: values is used to get input fiels data.

        Syntax: 
                element.value
                element.value = value

7] nodeValue

Definition: nodeValue gets or sets the value of a text node.

        Syntax: 
                element.nodeValue

9] getAttribute()

Definition: getAttribute() reads the value of an HTML attribute.

        Syntax: 
                element.getAttribute(attributeName) //attributeName → string

10] setAttribute()

Definition: setAttribute() sets or updates an attribute.

        Syntax: 
                element.setAttribute(attributeName) //attributeName → string
