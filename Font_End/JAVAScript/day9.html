Day 9: Advanced Array & Object Methods

forEach • map • filter • find • reduce • sort

1️⃣ What are Advanced Array Methods?

Definition: Built-in JavaScript methods to process data efficiently without manual loops.

array.method(callback)

2️⃣ forEach()

Definition: Executes a function once for each array element.

array.forEach((value, index) => { })

3️⃣ map()

Definition: Creates a new array by transforming each element.

array.map(value => newValue)

4️⃣ filter()

Definition: Filters elements based on a condition.

array.filter(value => condition)

5️⃣ find() & findIndex()

Definition: Finds the first matching value or index.

6️⃣ reduce()

Definition: Reduces array into a single value.

array.reduce((acc, val) => acc, initialValue)

7️⃣ sort()

Definition: Sorts array elements.
